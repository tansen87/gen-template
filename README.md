### 运行环境

- Node.js 16.14+

- pnpm 8.x+

- Python3.8-3.11

### 截图
![image](https://github.com/tansen87/gen-template/assets/98570790/a611d9ca-383a-415d-85ff-ecbe66ba943f)

### 初始化

```shell
pnpm run init
```

### dev

```shell
pnpm run py:dev
```

### build

```shell
# no console window
pnpm run build

# with console window
pnpm run pre
```
